# Virtual Barbershop

Progetto VR di barbershop virtuale sviluppato con Unity per Meta Quest.

## Struttura

```
VirtualBarbershop/   # Progetto Unity
├── Assets/          # Asset del progetto
├── Packages/        # Package Unity
└── ProjectSettings/ # Impostazioni progetto
```

## Requisiti

- Unity 2022.x o superiore
- Meta Quest SDK
- Android Build Support
